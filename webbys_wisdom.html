How the "n-th-of-type" Rule Makes Quokkaville Look Great!

Hey there, bright-eyed coders! Have you ever visited our We Are The Residents page and noticed how some quokka pictures float to the left while others float to the right? That’s not magic — it’s CSS using a clever trick called :nth-of-type()!

Here’s how it works: imagine you have a bunch of paragraphs (<p>) on your page, each with a photo inside. CSS can count them — 1st, 2nd, 3rd, and so on — and give each group its own style.

The code tells the even-numbered bios to float their images to the left, and the odd ones to the right. The result? A nice zigzag layout that’s fun to read and easy on the eyes!

It’s like a dance of images — left, right, left, right — across Quokkaville’s smiling faces. Try it on your own page and see how your stories come alive with rhythm and balance. Even the mayor says it’s picture-perfect!

CSS: Making Websites Look Awesome!

Have you ever seen a website with bright colors, fun fonts, or pictures in the perfect spot? That’s all thanks to CSS—the magic that makes websites look cool! CSS stands for Cascading Style Sheets, but don’t worry, it’s easier than it sounds.

Think of HTML as the skeleton of a webpage. CSS is like the clothes, paint, and decorations that make it stylish. With CSS, you can change colors, make text bigger or smaller, move pictures to the left or right, and even add fun animations!

You use selectors to tell the computer what to style. For example, p { color: blue; } turns all paragraphs blue, and h1 { font-size: 50px; } makes headings huge!

CSS is like being a web designer—you get to make your page look exactly how you imagine. Grab some code, get creative, and watch your webpage shine!

October 31st, 2025

HTML: The Magic Behind Webpages!

Have you ever wondered how websites show pictures, words, and buttons? That’s all thanks to HTML—it’s like the building blocks of the internet! HTML stands for HyperText Markup Language, but don’t let that scare you. Think of it as a giant LEGO set for web pages.

With HTML, you can tell the computer: “Hey! Make this a heading!” or “Put this picture here!” You do this using tags, which are special words inside < > symbols. For example, <p> starts a paragraph and <h1> makes a big heading. You can even add links, images, lists, and more!

HTML is super fun because it lets you create your own mini webpages. You can make a page about your favorite animals, hobbies, or even your dream city. It’s like painting—but with code!

So grab your keyboard, start typing, and let HTML help you build your own online world!
